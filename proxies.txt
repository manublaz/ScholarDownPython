# Archivo de configuración de proxies para ScholarDown Advanced

# IMPORTANTE: Los proxies se validan automáticamente antes de usar
# El sistema detecta automáticamente:
#   ✓ Proxies funcionales
#   ✗ Proxies que requieren autenticación
#   ✗ Proxies que no responden
#   ✗ Proxies bloqueados
#
# Ejemplos de formato correcto:
# 192.168.1.100:8080
# 10.0.0.50:3128
# 203.142.69.67:8080

# 2. Agrega tus proxies reales, uno por línea

# 4. Ejecuta 'python proxy_validator.py' para validar los proxies
# 5. El script rotará automáticamente entre proxies válidos

# RECOMENDACIONES:
# • Usar al menos 10-20 proxies para mejor rotación
# • Evitar proxies gratuitos públicos (suelen estar bloqueados)
# • Preferir proxies HTTP/HTTPS (no SOCKS)
# • Verificar que no requieran usuario/contraseña
# • Usar proxies de diferentes países/proveedores

# TIPOS DE PROXY SOPORTADOS:
# ✓ HTTP transparente (puerto 80, 8080, 3128)

# ✗ Proxies con autenticación (usuario:password)
